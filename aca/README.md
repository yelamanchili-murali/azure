# Azure Container Apps Projects

This repository contains various projects demonstrating Azure Container Apps capabilities and features.

## Projects

### [Session Affinity Demo](./session-affinity/)

Demonstrates session affinity (sticky sessions) in Azure Container Apps using a Spring Boot application. Includes:
- Local testing with Nginx load balancer
- Azure Container Apps deployment with session affinity configuration

### [VNet and Workload Profile Demo](./vnet-workloadprofile/)

Demonstrates Azure Container Apps deployment with Virtual Network integration and custom Workload Profiles. Features:
- VNet integration for secure networking
- Custom Workload Profiles for optimized resource allocation
- Infrastructure as Code deployment using Azure CLI
